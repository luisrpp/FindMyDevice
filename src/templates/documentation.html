FindMyDevice API

  • Class name: DeviceLocationHandler
  • uri: /api/location
  • Format: json
  • Methods:
    • create: to set a new device location.
      • device_name: name of the registered device.
      • device_id: mobile id of the generated device.
      • lat: latitude
      • lon: longitude

You can try it with curl:

$ curl -u <USER>:<PASSWORD> "http://findmydevice-luisrpp.herokuapp.com/api/location/?format=json" \
-F "device_name=<YOUR_DEVICE>" -F "device_id=<MOBILE_ID>" -F "lat=<LAT>" -F "lon=<LON>"
